# 데뷰 영어학원 웹사이트

초1~중3 대상의 데뷰 영어학원 디데이 카운터 웹사이트입니다.

## 주요 기능

- 2025년 5월 24일까지의 디데이 카운터
- 학원 소개 및 특징 안내
- 상담 및 문의 링크 (카카오톡, 블로그, 밴드, 위치)

## 기술 스택

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## 로컬 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```
